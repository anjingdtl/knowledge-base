# 知识图谱功能介绍

## 概述

ShineHeKnowledge 的知识图谱功能帮助你可视化和探索知识条目之间的关系。

## 图谱类型

### 文件大纲图谱
每个文档导入后会自动解析其结构（标题层级、段落），形成文件内部的大纲图谱。

### 双向链接
使用 `[[页面标题]]` 语法可以在知识条目之间建立双向链接。系统会自动检测并记录这些关系。

### 标签关系
通过标签系统，你可以将相关知识条目归类。标签之间也可以建立层级关系（父子关系）。

## 图谱可视化

在 GUI 的"图谱"页面，你可以看到以力导向布局展示的知识网络：

- **节点颜色**：按文件类型区分（Markdown、PDF、Word 等）
- **边类型**：表示不同的关系类型（包含、引用、标签关联等）
- **交互操作**：支持缩放、拖拽、点击查看详情

## 图谱存储

ShineHeKnowledge 使用 SQLite 内置图索引：

- Page、Block、Tag、引用关系和语义关系都保存在本地 `data/kb.db`
- 无需部署外部图数据库服务
- 统一图谱和多跳遍历从 SQLite 表动态构建

## 图查询

通过 MCP 工具或 API，你可以执行图遍历查询：

- 从指定节点出发的多跳遍历
- 按关系类型过滤
- 查询两个节点之间的路径
