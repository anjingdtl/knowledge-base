# ShineHeKnowledge — 本地优先的智能知识库

## 什么是 ShineHeKnowledge？

ShineHeKnowledge 是一个完全本地运行的知识库系统，支持多模态文档管理、RAG 智能问答和知识图谱构建。你的数据永远不会离开你的电脑。

## 核心功能

### 📚 多格式文档导入
支持 PDF、Word、Excel、Markdown、HTML、代码文件等 30+ 种格式的一键导入。导入后自动完成文本分块、向量化和索引构建。

### 🔍 RAG 智能问答
结合向量搜索和大语言模型（LLM），你可以用自然语言向知识库提问。系统会自动检索最相关的知识片段，生成精准的回答。

### 🔗 知识图谱
自动构建文档之间的关系网络。支持 `[[双链]]` 语法，让知识条目之间互相关联。可视化图谱帮助你对知识全局有清晰认知。

### 🤖 MCP 协议集成
通过 Model Context Protocol (MCP)，ShineHeKnowledge 可以与 Claude、Cursor、Cline 等 AI 编码工具无缝连接，让你的 AI 助手直接访问你的知识库。

## 三种运行模式

| 模式 | 说明 |
|------|------|
| **桌面 GUI** | PySide6 暗色主题，适合日常使用 |
| **REST API** | FastAPI 服务，适合集成到其他应用 |
| **MCP Server** | 暴露 ~30 个工具，供 AI 编码工具调用 |

## 快速开始

1. 打开聊天界面，输入你的第一个问题
2. 尝试导入一些文档到知识库
3. 在设置中配置更多 AI 服务选项
