MCP Server 的工具是怎么按配置档注册的？
MCP Server 通过 src/mcp/tool_registry.py 按配置档（profile）声明式注册工具[1]。当前支持 core / extended / admin / full / legacy 五种配置档，默认使用 extended，暴露 20 个工具[2]。核心检索工具（ping、search、ask、read 等 10 个）在所有配置档中始终可用；Wiki、图谱、Agent Memory 等实验性工具需开启 experimental_tools_enabled[3]。
