# Database Systems Overview

## PostgreSQL

PostgreSQL is an advanced open-source relational database. It supports JSON, full-text search via tsvector, and extensible indexing with GiST and GIN.

## MongoDB

MongoDB is a document-oriented NoSQL database. It stores data as BSON documents and supports Atlas Search for full-text capabilities.

## Elasticsearch

Elasticsearch provides distributed search and analytics. It uses inverted indices and BM25 scoring for relevance ranking.

None of these databases are used as the primary storage in this system.
