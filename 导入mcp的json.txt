============================================================
  ShineHeKnowledge MCP Server 导入配置指南
============================================================

用途
----
本项目主要面向 Agent 通过 MCP 调用本地知识库。推荐优先使用 stdio 模式：
由 Claude Code、Cursor、Cline、Roo Code、Windsurf、TeleClaw 等 Agent 自动启动
run_mcp.py，不需要手动常驻服务，稳定性最高。

当前项目路径
------------
D:/ClaudeCodeWorkSpace/projects/knowledge-base

推荐 Python 路径
---------------
C:/Users/Administrator/AppData/Local/Programs/Python/Python314/python.exe

Windows 关键注意事项
--------------------
1. 不要用 cmd /c 包装 MCP 命令；cmd 会占用 stdin，破坏 MCP JSON-RPC 握手。
2. 推荐使用 python.exe 的绝对路径，不要依赖 Agent 子进程里的 PATH。
3. 必须设置 cwd 或 SHINEHE_HOME，确保 config.yaml、data/kb.db 能被正确找到。
4. stdio 模式不需要手动执行 start_mcp.bat；Agent 会按 JSON 配置自动启动。
5. HTTP 模式只用于多个 Agent 共享一个常驻服务，需先运行 start_mcp.bat。

Claude Code 快速导入
--------------------
方式一：源码 stdio 模式，推荐

claude mcp add-json shinehe-kb "{\"command\":\"C:/Users/Administrator/AppData/Local/Programs/Python/Python314/python.exe\",\"args\":[\"D:/ClaudeCodeWorkSpace/projects/knowledge-base/run_mcp.py\"],\"cwd\":\"D:/ClaudeCodeWorkSpace/projects/knowledge-base\",\"env\":{\"SHINEHE_HOME\":\"D:/ClaudeCodeWorkSpace/projects/knowledge-base\"},\"type\":\"stdio\"}" -s user

方式二：已执行 pip install -e . 后

claude mcp add shinehe-kb -- shinehe-mcp

方式三：HTTP 模式，需先双击 start_mcp.bat 或执行下列命令

python run_mcp.py -t streamable-http --host 127.0.0.1 --port 9000
claude mcp add -t http shinehe-kb http://127.0.0.1:9000/mcp

通用 mcpServers JSON
--------------------
把下面片段放入支持 mcpServers 的 Agent 配置文件：

{
  "mcpServers": {
    "shinehe-kb": {
      "command": "C:/Users/Administrator/AppData/Local/Programs/Python/Python314/python.exe",
      "args": [
        "D:/ClaudeCodeWorkSpace/projects/knowledge-base/run_mcp.py"
      ],
      "cwd": "D:/ClaudeCodeWorkSpace/projects/knowledge-base",
      "env": {
        "SHINEHE_HOME": "D:/ClaudeCodeWorkSpace/projects/knowledge-base"
      },
      "type": "stdio"
    }
  }
}

OpenCode JSON
-------------
OpenCode 使用 mcp 字段，不是 mcpServers：

{
  "mcp": {
    "shinehe-kb": {
      "type": "local",
      "command": [
        "C:/Users/Administrator/AppData/Local/Programs/Python/Python314/python.exe",
        "D:/ClaudeCodeWorkSpace/projects/knowledge-base/run_mcp.py"
      ],
      "environment": {
        "SHINEHE_HOME": "D:/ClaudeCodeWorkSpace/projects/knowledge-base"
      },
      "enabled": true
    }
  }
}

HTTP 远程接入 JSON
------------------
先启动：
python run_mcp.py -t streamable-http --host 0.0.0.0 --port 9000

再配置：

{
  "mcpServers": {
    "shinehe-kb": {
      "type": "http",
      "url": "http://127.0.0.1:9000/mcp"
    }
  }
}

一键写入 Agent 配置
-------------------
也可以运行交互式脚本：

python scripts/setup_mcp.py

脚本会自动生成 stdio 配置，并写入所选 Agent 的配置文件。
