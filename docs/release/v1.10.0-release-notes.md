# ShineHeKnowledge v1.10.0 Release Notes

**Maintainability Closure** — formal architecture after the v1.9.x transition train.

## Highlights

- **Unified retrieval only** — Legacy/shadow orchestrator pipelines deleted after Unified default observation
- **Answer package** — assemble / citations / fallbacks under `src/answering/`
- **MCP domain tools** — `server.py` ≤500 lines; tools in `src/mcp/tools/*`
- **Container Providers** — Core / Verified / Authoring / Experimental own construction & close
- **Migration head gate** — write-mode boot refused when schema behind Alembic head
- **KnowledgeTagRepository** — auto_tag path has no embedded SQL in application layer
- **CI gates** — `architecture-closure`, `migration-gate`, `contract-gate`
