# ShineHeKnowledge v1.9.0 — Maintainability Phase 3

## Highlights

- **Unified Answer orchestration:** `src/answering/` with `AnswerExecution` + `AnswerService`; Ask goes through SearchExecution only.
- **Answer cutover modes:** `answer.orchestrator` = `legacy` | `shadow` | `unified` (default unified; structural parity, not full LLM text equality).
- **MCP structure:** runtime/auth/envelopes/policies extracted; implementation in `src/mcp/server.py`; `src/mcp_server.py` remains a compatibility alias.
- **Container capability groups:** `container.groups.core|verified|authoring|experimental` without removing flat lazy properties.
- **Architecture gates:** forbid retrieval/answering → mcp and repositories → container imports.
- **Database migration freeze:** `_migrate()` statement inventory frozen; new schema must use Alembic.
- **Deprecation register:** `docs/migration/deprecation-register.md`.
