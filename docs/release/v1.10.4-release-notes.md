v1.10.4 是 **生产试点最终验收基础设施 + 检索/路由/稳定性质量修复** 发布。  
建立人工 Ground Truth 评估体系，修复假拒答、结果去重、数字单位、结构化路由与 Provider 超时隔离等问题。

**无数据库 Schema 变更，无 Alembic Revision。** 公开工具名与主要 MCP 配置档语义保持兼容。

> **生产试点结论：** 本版本**未声称**达到生产试点门槛。Precision@5 / nDCG@10 / Numeric 仍未达标；可继续受控内测。

## 改动摘要

### 生产试点评估体系

- 人工 Ground Truth 数据集：retrieval / no_answer / numeric / routing / answer_citations
- `evals/production_pilot_metrics.py`：指标分母与 scope 校正（禁止空结果默认满分）
- Harness：`scripts/production_pilot_eval.py`、`scripts/production_pilot_mcp_harness.py`
- 基线冻结：`artifacts/production-pilot-final-validation/baseline.json`

### 检索与问答质量

- **结果去重**：`dedupe_by_knowledge_id`，提升 Precision 相关排序纯度
- **复合数字单位**：`珠/米` 等复合单位抽取与降权，改善 Numeric Top1
- **标题强证据**：库内标题命中时降低假 no-answer
- **标题词项 boost** + structured 参数提升

### 路由

- `file_type` 结构化规则（列举全部 md/pdf 等）
- hybrid 分析意图优先于偶然 tag structured
- 图优先路由稳定性测试加固

### 稳定性与路径

- Provider 非协作调用在 worker 进程中可终止
- 超时风暴后 isolation 状态可重置
- 正式向量索引路径解析统一
