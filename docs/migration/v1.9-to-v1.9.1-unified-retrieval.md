# v1.9.0 → v1.9.1：Unified Retrieval 默认

> **状态：** 随 maintainability closure WP1 交付（Unified 默认；Legacy 保留回滚）

## 变更摘要

| 项 | v1.9.0 | v1.9.1 |
|---|---|---|
| `retrieval.orchestrator` 默认 | `legacy` | **`unified`** |
| Raw 算法权威 | SearchService 私有方法 / 适配器 | `RawRetriever` |
| Verified 融合 | `SearchService._search_verified_hybrid` | `VerifiedFusion` + Policy 组合 |
| Legacy 回滚 | 默认主路径 | 配置 `retrieval.orchestrator: legacy` |

## 配置

```yaml
retrieval:
  orchestrator: unified   # 默认；正式路径
  # orchestrator: legacy  # 紧急回滚
  # orchestrator: shadow  # 双跑对比（返回 legacy 结果，记录 unified 差异）
```

省略 `retrieval` 块时按 **unified** 运行。

## 回滚

1. 配置：`retrieval.orchestrator: legacy`
2. 代码：`src/compatibility/legacy_retrieval.py` 仍提供主路径
3. 无数据库 Schema 变更
